# Valorant RANKME
**Valorant RANKME es una forma de conocer tu rendimiento en diversos modos de juego de Valorant.**  
(Valorant es un videojuego de estrategia, trabajo en equipo y disparos en primera persona)
## ¿Por qué es interesante?
- Valorant cuenta con muchos modos de juego, pero solo para unos pocos le proporciona un rango a los jugadores.
- Valorant RANKME intenta llenar ese vacío haciendo que para cada modo de juego disponible los jugadores puedan tener un feedback en base a cómo están jugando.
## ¿Cómo funciona por detrás?
- Cada vez que un usuario solicita ver su rendimiento en la página, el mismo se añade a la base de datos.
- Cuando el jugador es registrado en la base de datos, se consigue información de su rendimiento y últimas partidas en el juego.
- El servidor solicita los usuarios registrados y les hace un proceso de tracking.
- Durante el proceso, cada cierto tiempo revisa si hubo una nueva partida, y si la hay actualiza la información del jugador en la base de datos.
- Eso incluye: todas las nuevas partidas, rangos según modo de juego y temporada, e información cosmética como tarjetas o íconos.
